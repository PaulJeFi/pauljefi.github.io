Courbes implicites

Sphère

La sphère a pour équation \(\displaystyle \left(x-\omega_x\right)^2+\left(y-\omega_y\right)^2+\left(z-\omega_z\right)^2 = R^2\) où \(\displaystyle \Omega(\omega_x, \omega_y, \omega_z)\) est le centre et \(\displaystyle R\) le rayon.


Cône

Le cône a pour équation \(\displaystyle \left(x-\omega_x\right)^2+\left(y-\omega_y\right)^2 = \left(z-\omega_z\right)^2\) où \(\displaystyle \Omega(\omega_x, \omega_y, \omega_z)\) est le sommet.



Cylindre

Le cylindre a pour équation \(\displaystyle \left(x-\omega_x\right)^2+\left(y-\omega_y\right)^2 = R^2\) où \(\displaystyle \Omega(\omega_x, \omega_y)\) est le centre et \(\displaystyle R\) le rayon.



Hyperboloïde

L'hyperboloïde a pour équation \(\displaystyle \left(x-\omega_x\right)^2+\left(y-\omega_y\right)^2-\left(z-\omega_z\right)^2 = R^2\) où \(\displaystyle \Omega(\omega_x, \omega_y, \omega_z)\) est le centre et \(\displaystyle R\) le rayon.



Ellipsoïde

L'ellipsoïde a pour équation \(\displaystyle \frac{\left(x-\omega_x\right)^2}{a^2}+\frac{\left(y-\omega_y\right)^2}{b^2}+\frac{\left(z-\omega_z\right)^2}{c^2} = 1\) où \(\displaystyle \Omega(\omega_x, \omega_y, \omega_z)\) est le centre et \(\displaystyle a\), \(\displaystyle b\) et \(\displaystyle c\) les demi-axes.



Paraboloïde hyperbolique

Le paraboloïde hyperbolique a pour équation \(\displaystyle z = \left(\frac{x}{a}\right)^2-\left(\frac{y}{b}\right)^{2}\).
